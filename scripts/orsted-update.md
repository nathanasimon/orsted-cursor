# Orsted Update

Update Orsted to the latest version.

## What This Does

1. Checks if Orsted is installed
2. Pulls latest changes from the repository
3. Reinstalls with the updated files
4. Prompts you to restart Cursor

## Usage

Run this command to update Orsted when new versions are available.

## Manual Update

You can also update manually:

```bash
cd /path/to/orsted-cursor-repo
git pull
./install.sh
```

Then restart Cursor.
